Dependencies
------------

You just need the [SteveHand](http://sjfonts.sourceforge.net/) font. On a
Debian/Ubuntu client:
```
$ sudo apt-get install ttf-sjfonts
```

On a web server you have to provide this font as `fonts/SteveHand.woff`.
